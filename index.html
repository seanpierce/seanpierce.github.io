MISSISSIPPIRECORDS.STORE is a business-to-business e-commerce site for beloved Portland-based record store, label, distro Mississippi Records. Developed using PHP and JavaScript, this project employs creative solutions to address the shop's specific and unique needs. Currently being recreated as a rails application.

SOUNDS_COOL is a browser-based, fully analog-modeled synthesizer with sequencer. Developed using Web Audio API, a user has full control over the synthesizer's pitch, filter, envelope and modulation parameters. The sequencer is connected to a custom made API which can store and recall a user's patterns on the fly.

INTERNET NOISE is a javascript application that floods an ISP's database with nonsense by visiting random websites and performing random Google searches.

JOIN IN THE CHAT is a simple ruby on rails application that utilizes action cables to handle multiple connection instances. Messages are instantly rendered to the users asynchronously. Users can connect to the application from anywhere and engage in the conversation.

DOCTOR FINDER API uses the Google Maps API in conjunction with the Better Doctor API to allow users to retrieve information for doctors in a specified location who treat a specified symptom. I implemented an easy to use interface with an elegant and uncomplicated design.

INTROTORHYTHM.COM is a freeform podcasting series which features guest mixes and live recordings that represent a wide variety of styles. Key features include custom javascript audio player with controls, dynamic and RESTful page routing for each podcast, endless looping of all podcasts, and a custom subscription form.
